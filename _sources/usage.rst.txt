=====
Usage
=====

Start by importing HiCap_Analysis

.. code-block:: python

    import hicap_analysis


Or import a well functions

.. code-block:: python

    import hicap_analysis.wells as wo

    ddwn = wo._theis()

Or import a specific function.

.. code-block:: python

    from hicap_analysis.wells import _theis

    ddwn = _theis()
