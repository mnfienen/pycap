=====
Usage
=====

Start by importing `pycap`

.. code-block:: python

    import pycap


Or import a well functions

.. code-block:: python

    import pycap.wells as wo

    ddwn = wo.theis()

Or import a specific function.

.. code-block:: python

    from pycap.wells import glover

    depl = glover()
